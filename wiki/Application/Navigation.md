<h1>General Navigation</h1>
<p>
    The application is designed to be a typical Windows application. If the user is familiar with using Windows
    within normal circumstances, they should
    have little to no problems navigating the application without assistance. There are extra features included
    that the user may not be aware of. This
    documentation will help the user become more familar with how to navigate the application as well as use the
    more advanced navigational features.
</p>

<img src="~/img/navigation_top.jpg" class="img-fluid" alt="What the main list view screen looks like on Entities
     Maintenance screen." />
<p>
    The application can be navigated on most screens using the keyboard. This includes the arrow keys, exit key,
    &amp; return key. There are also hotkeys for certain functions, which are typically listed on those specific
    actions
</p>
